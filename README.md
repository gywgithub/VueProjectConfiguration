

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification (release branch)
npm run build

# build for production with minification (master branch)
npm run build-master

# serve with hot reload at localhost:23005
npm run server

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
